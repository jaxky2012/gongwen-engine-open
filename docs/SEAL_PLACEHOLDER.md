# 演示印章说明

本目录包含演示用的占位印章图片，仅供测试使用。

## 文件说明

- `seal_demo.png` - 演示印章（带"仅供测试"水印）
- `logo_demo.png` - 演示红头Logo

## 使用说明

将这些图片替换为您企业的正式印章和Logo。

### 印章规范

- 格式：PNG（透明背景）
- 尺寸：建议 4.5cm × 4.5cm
- 分辨率：300 DPI 以上

### 红头Logo规范

- 格式：PNG（透明背景）
- 高度：建议 1.2cm
- 分辨率：300 DPI 以上

## 安全提醒

**切勿将正式印章图片上传到公开仓库！**
